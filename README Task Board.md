# 📋 Kanban Task Board

تطبيق ويب بسيط لإدارة المهام باستخدام نظام Kanban، مبني بـ HTML و CSS و JavaScript بدون أي مكتبات خارجية.

## المميزات

- إضافة مهام جديدة بسهولة
- سحب وإفلات المهام بين الأعمدة الثلاثة
- تحديد المهمة كمكتملة عند نقلها لعمود Done
- حذف أي مهمة بزر ✕
- عداد للمهام في كل عمود

## الأعمدة

| العمود | الوصف |
|--------|--------|
| To Do | المهام المطلوب تنفيذها |
| In Progress | المهام قيد التنفيذ |
| Done | المهام المكتملة |

## تشغيل المشروع

لا يحتاج إلى أي تثبيت — فقط افتح الملف في المتصفح:

```bash
open index.html
```

أو من خلال VS Code Live Server.

## هيكل الملفات

```
kanban-task-board/
└── index.html       # كل الكود (HTML + CSS + JS) في ملف واحد
```

## التقنيات المستخدمة

- HTML5 (Drag and Drop API)
- CSS3 (animations, flexbox, custom properties)
- Vanilla JavaScript (no frameworks)

## لقطة شاشة

> افتح `index.html` في المتصفح لترى التطبيق.
